# kgbook-2020

当前代码为图书 《自然语言理解与行业知识图谱-概念、方法与工业落地》 中介绍的各个章节的算法展示代码。
相关项目以python语言开发，提供了常见的机器学习算法及样例代码。
提供了文本分类、文本标注、文本语义匹配、文本生成、文本聚类等多任务介绍、自制数据集、代码库。
提供了知识图谱构建的样例代码和示例文件；
提供了行业知识图谱从0到1的开发过程，相关代码，数据集和应用效果。

我们也希望能够帮助各个行业做认知赋能，当然这仅仅是第一步！
